# FCalc

A lightweight Windows desktop calculator built with C# and WinForms.

---

## Features

- Basic arithmetic — addition, subtraction, multiplication, and division
- Square root
- Decimal input
- Negate (flip positive/negative)
- Backspace and clear
- Rounded button UI
- First-run welcome message

## Requirements

- Windows OS
- .NET Framework (WinForms)

## How to Use

1. Open FCalc
2. Enter a number using the on-screen buttons
3. Press an operator (`+`, `-`, `×`, `÷`)
4. Enter the second number
5. Press `=` to see the result

Other buttons:
| Button | Action |
|--------|--------|
| `C` | Clear everything |
| `DEL` | Delete last digit |
| `.` | Add decimal point |
| `±` | Negate the current number |
| `√` | Square root of current number |

## Notes

- Dividing by zero will show an error message rather than crash
- FCalc saves a small registry entry (`SOFTWARE\FCalc`) on first launch to show a one-time welcome message
